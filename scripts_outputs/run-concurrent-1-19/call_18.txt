Okay, her er et program, der læser afstanden fra en afstandssensor tilsluttet port C og udskriver den. Jeg har kommenteret koden grundigt på dansk, så du kan forstå, hvad hver linje gør.

```python
# Importerer modulen, der styrer afstandssensoren
import distance_sensor
# Importerer port-konstanterne for at referere til port C tydeligt
from hub import port

# Definerer porten, hvor afstandssensoren er tilsluttet
DISTANCE_PORT = port.C

# En uendelig løkke, der kontinuerligt læser og udskriver afstanden
while True:
    # Læser afstanden fra sensoren på port C.
    # Værdien returneres i millimeter.
    # Hvis der ikke er noget inden for rækkevidde, returneres -1.
    distance_mm = distance_sensor.distance(DISTANCE_PORT)

    # Udskriver afstanden til konsollen.
    # Dette giver dig mulighed for at se den aktuelle afstand.
    print(f"Afstand: {distance_mm} mm")

    # Pauser programmet i 100 millisekunder.
    # Dette forhindrer, at programmet kører for hurtigt og overbelaster systemet.
    # Det giver også dig mulighed for at se resultaterne i realtid.
    import time
    time.sleep_ms(100)
```

**Forklaring:**

*   **`import distance_sensor`**:  Dette importerer `distance_sensor`-modulet, så du kan bruge funktioner som `distance()`.
*   **`from hub import port`**: Dette importerer `port`-konstanterne, så du nemt kan referere til port C.
*   **`DISTANCE_PORT = port.C`**:  Dette definerer en variabel `DISTANCE_PORT`, der holder port C, hvilket gør koden mere læsbar.
*   **`while True:`**: Dette starter en uendelig løkke, så programmet fortsætter med at læse afstanden.
*   **`distance_mm = distance_sensor.distance(DISTANCE_PORT)`**:  Dette kalder funktionen `distance()` fra `distance_sensor`-modulet og sender `DISTANCE_PORT` (port C) som argument.  Den returnerede afstand (i millimeter) gemmes i variablen `distance_mm`.
*   **`print(f"Afstand: {distance_mm} mm")`**: Dette udskriver afstanden til konsollen. `f-string` bruges til at formatere outputtet, så det er let at læse.
*   **`import time` og `time.sleep_ms(100)`**: Dette importerer `time`-modulet og bruger `sleep_ms` til at pause programmet i 100 millisekunder.  Dette forhindrer programmet i at overbelaste systemet og giver dig mulighed for at se resultaterne i realtid.

**Sådan bruges koden:**

1.  Tilslut din afstandssensor til port C på SPIKE-hubben.
2.  Upload koden til SPIKE-hubben.
3.  Kør programmet.
4.  Programmet vil kontinuerligt udskrive afstanden fra sensoren til konsollen (f.eks. i SPIKE App).
