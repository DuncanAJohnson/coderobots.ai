Okay, her er et program der bevæger motorer på portene C og D. Jeg har inkluderet kommentarer på dansk, der forklarer hvad koden gør.

```python
# Importerer modulen som styrer motorene
import motor
# Importerer port-konstanterne fra hub modulen
from hub import port

# Konfigurerer hastigheden og varigheden for motorene.
# Juster disse værdier efter behov.
velocity = 100  # Grader pr. sekund
duration_ms = 2000  # 2000 millisekunder (2 sekunder)

# Starter motoren på port C.
motor.run(port.C, velocity)

# Starter motoren på port D.
motor.run(port.D, -velocity) # Negativ hastighed for at køre motoren i den modsatte retning.

# Vent i den specificerede varighed (ikke-blokerende).
# Tiden vil blive ventet i millisekunder.
import time
time.sleep_ms(duration_ms)

# Stopper motoren på port C
motor.stop(port.C)

# Stopper motoren på port D
motor.stop(port.D)

# Slut på programmet
```

**Forklaring:**

*   **`import motor`**:  Denne linje importerer `motor`-modulet, som giver dig adgang til funktioner til at styre SPIKE Prime motorer.
*   **`from hub import port`**:  Importerer `port`-konstanterne, så du nemt kan referere til portene A, B, C, osv.
*   **`velocity = 100`**:  Definerer en variabel `velocity` til at holde hastigheden for motorene.  Hastigheden er målt i grader pr. sekund.
*   **`duration_ms = 2000`**: Definerer en variabel `duration_ms` til at holde varigheden af motoren i millisekunder.
*   **`motor.run(port.C, velocity)`**:  Starter motoren på port C med den specificerede hastighed.
*   **`motor.run(port.D, -velocity)`**:  Starter motoren på port D med den specificerede hastighed.  Jeg har brugt en negativ hastighed for at få motoren til at køre i den modsatte retning.
*   **`import time` og `time.sleep_ms(duration_ms)`**:  Bruger `time`-modulet til at vente i den specificerede varighed. Dette er en ikke-blokerende funktion.
*   **`motor.stop(port.C)` og `motor.stop(port.D)`**: Stopper motorene på port C og D.

Husk at justere `velocity` og `duration_ms` for at opnå den ønskede bevægelse.  Du kan også ændre retningen af motoren på port D ved at ændre fortegnet for `velocity`.
